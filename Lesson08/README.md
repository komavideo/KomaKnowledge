## 实战讲解

### 安装

```bash
$ sudo npm install -g preact-cli
$ preact -h
# 列出可用模板
$ preact list
```

### 项目实战

```bash
# 新建项目
$ preact create default myweb
$ cd myweb
# 安装依赖包
$ npm install

# 项目编辑
$ code .

# 开发模式：启动侦听服务
$ npm run dev
# 产品模式：打包应用程序，生成静态build目录
$ npm run build
```
